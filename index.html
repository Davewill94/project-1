How to play

  • To start playing press the start button just below the color tiles.
  • Watch the color pattern and repeat it back. Every 5 turns the color speeds will increase.
  • If the sounds annoy you there is a mute button.
  • If you want to quit just hit the end game button.
  • If you need a little more of a challenge, look below.
